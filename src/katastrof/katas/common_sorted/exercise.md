# Common Sorted

Implement `common_sorted(left: list[str], right: list[str]) -> list[str]`.

Both input lists are sorted. Return the common values while preserving duplicate counts.

Example:

```python
common_sorted(["a", "e", "e", "e"], ["b", "b", "c", "e", "e", "g"]) == ["e", "e"]
```

Aim for `O(n + m)` time.
