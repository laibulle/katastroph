# Move Dots

Implement `move_dots_to_end(items: list[str]) -> list[str]`.

Move every `"."` to the end of the list while preserving the relative order of every
other value.

Example:

```python
move_dots_to_end(["a", "b", ".", "c", ".", ".", "k"])
```

returns `["a", "b", "c", "k", ".", ".", "."]`.
